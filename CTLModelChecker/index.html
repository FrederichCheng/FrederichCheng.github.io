State Transition System:
The STS must follow the format (Similar to Kripke Structure ):
{
{< state >,< state >, ...},
< initial state >,
{< state > -> < state >, < state > -> < state >, ...},
{< atomic symbol >:{< state >, < state >, < state >, ...},< atomic symbol >:{< state >, < state >, < state >, ...}, ...}
}

For example: (Mutual Exclusion for 2 processes)
{{s0, s1, s2, s3, s4, s5, s6, s7, s8, s9, s10, s11, s12}, s0 , {s0->s1, s1->s2, s1->s3,s2->s4, s3->s4, s3->s5, s4->s6, s5->s6, s5->s7, s6->s9, s7->s8, s7->s9, s8->s10, s9 -> s10, s9->s11, s10->s12, s11->s12, s11->s1, s12->s3}, { n0:{s0, s1, s2, s7, s9, s11}, n1:{s0, s1, s3, s5, s7, s8}, w0:{s3, s4, s8, s10, s12}, w1:{s2, s4, s6, s9, s10} , t0:{s1, s2, s3, s4, s5, s6}, t1:{s7,s8, s9, s10, s11, s12}, c0:{s5, s6}, c1:{s11, s12} } }

CTL Expression:
Unary Operators: AX, EX, AG, EG, AF, EF, !(not)
Binary Operators: & (AND), | (OR), -> (Condition), <-> (IFF), AU, EU
You can also have parenthesis.
For example: AF(c1&EXc1)
